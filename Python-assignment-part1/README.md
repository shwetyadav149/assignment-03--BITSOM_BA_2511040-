# Python Assignment — Part 1: Student Grade Tracker

## Description
This project implements a command-line Student Grade Tracker using Python basics.

## Tasks Covered
- Data cleaning and parsing
- Name validation
- Marks analysis (total, average, grades)
- Class performance summary
- String manipulation

## Features
- Converts raw data into structured format
- Calculates grades and averages
- Handles user input with validation
- Displays formatted output

## How to Run
1. Open the notebook file
2. Click "Run All"
3. View outputs in each cell
